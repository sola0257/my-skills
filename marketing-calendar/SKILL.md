# 营销日历工具 v1.0

## 📋 概述

本工具用于查询绿植园艺行业的营销热点，帮助运营者把握时间节点，提前规划内容。

**核心功能：**
1. **每日查询**：查询指定日期的节气、节日、园艺逻辑、内容钩子。
2. **日历生成**：基于模板自动生成指定年份的完整营销日历。

**数据来源：**
- `/Users/dj/Desktop/全域自媒体运营/实操手册/营销日历-[年份].md`

## 🔄 执行流程

### 1. 查询模式

**命令**：`/marketing-calendar [日期]` (日期可选，默认为今天)

**逻辑**：
1. 读取对应年份的 markdown 日历文件。
2. 查找匹配日期的行或最近的节气/节日。
3. 提取：园艺逻辑、社媒钩子、变现策略。

**输出示例**：
```markdown
📅 **2026年1月24日 营销建议**

🌱 **园艺逻辑**：深度冬季养护期，植物休眠，减少浇水。

🎯 **临近节点**：
- **1月29日（大寒/年宵花高峰）**：还有 5 天
  > 建议：现在是推年宵花的最佳时机！

📱 **内容建议 (公域)**：
- 🎥 Vlog：春节前的花园大扫除
- 📝 干货：年宵花买回家怎么养不黄叶？

💰 **变现建议 (私域)**：
- 🛒 团购：年宵花（蝴蝶兰/冬青）最后冲刺
- 🎁 礼包：春节送礼绿植套餐
```

### 2. 生成模式

**命令**：`/marketing-calendar --generate [年份]`

**逻辑**：
1. 使用内置的节气算法和固定节日表。
2. 结合《绿植园艺全域运营实操手册》中的营销逻辑。
3. 生成完整的 Markdown 表格文件。
4. 保存至 `/Users/dj/Desktop/全域自媒体运营/实操手册/`。
