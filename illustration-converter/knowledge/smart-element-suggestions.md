# 智能元素建议系统

> **用途**：根据植物特性自动推荐合适的意境图元素
> **目标**：提供专业、合理、有创意的元素建议

---

## 🎯 推荐逻辑

### 基于植物类型的推荐

#### 1. 开花植物

**特征**：有明显的花朵、色彩鲜艳

**推荐元素**：
- **人物**：小女孩欣赏花朵、园丁修剪花枝
- **动物**：蝴蝶飞舞、蜜蜂采蜜、小鸟停留
- **物品**：花瓶、园艺剪刀、浇水壶

**推荐理由**：开花植物自然吸引授粉昆虫，人物欣赏花朵符合常理

**示例**：
```
植物：金鱼草（开花植物）
推荐：
- 人物：小女孩好奇地观察花朵
- 动物：蝴蝶在花朵周围飞舞
- 物品：精致的陶瓷花瓶
```

---

#### 2. 观叶植物

**特征**：叶片为主要观赏部分、形态独特

**推荐元素**：
- **人物**：园丁照料、古装人物赏叶
- **动物**：小鸟栖息、蜻蜓停留
- **物品**：书本、茶具、文房四宝

**推荐理由**：观叶植物适合营造宁静、文雅的氛围

**示例**：
```
植物：龟背竹（观叶植物）
推荐：
- 人物：古装人物在植物旁品茶
- 动物：小鸟栖息在叶片上
- 物品：茶具、书本
```

---

#### 3. 多肉植物

**特征**：肉质叶片、小巧可爱

**推荐元素**：
- **人物**：小女孩浇水、园丁照料
- **动物**：小蜜蜂（如果开花）、蝴蝶
- **物品**：可爱的花盆、园艺工具、小铲子

**推荐理由**：多肉植物小巧可爱，适合童趣、治愈系元素

**示例**：
```
植物：桃蛋（多肉植物）
推荐：
- 人物：小女孩轻轻浇水
- 动物：小蝴蝶飞过
- 物品：可爱的陶瓷小花盆、小浇水壶
```

---

#### 4. 藤蔓植物

**特征**：枝条蔓延、垂吊或攀爬

**推荐元素**：
- **人物**：园丁修剪、小女孩观察
- **动物**：小鸟在藤蔓间穿梭、蝴蝶飞舞
- **物品**：支架、花架、园艺工具

**推荐理由**：藤蔓植物的动态感适合飞行的动物

**示例**：
```
植物：绿萝（藤蔓植物）
推荐：
- 人物：园丁修剪枝条
- 动物：小鸟在藤蔓间飞舞
- 物品：花架、修剪工具
```

## ⚠️ 推荐原则

1. **符合植物特性**：元素应与植物的生态特征相符
2. **符合画风特点**：元素应与画风的文化背景和审美特点一致
3. **符合用户需求**：考虑用户的使用场景和情感表达需求
4. **保持专业性**：推荐应基于专业的园艺和艺术知识
5. **提供选择**：给用户多个选项，而不是单一答案
