## 使用说明

### 1. 创建正面案例

```bash
# 1. 复制模板
cp CASE_TEMPLATE.md positive/case-001-watercolor-oriental-succulent.md

# 2. 填写内容
# 3. 保存图片到 .tmp/ 目录
# 4. 更新 INDEX.md
```

### 2. 创建反面案例

```bash
# 1. 复制模板
cp CASE_TEMPLATE.md negative/case-001-oversaturated-colors.md

# 2. 填写内容
# 3. 保存对比图片
# 4. 更新 INDEX.md
```

### 3. 更新索引

在 `INDEX.md` 中添加案例链接：

```markdown
### 清新水彩（东方）

- [案例 001：多肉植物桃蛋](./positive/case-001-watercolor-oriental-succulent.md) ⭐⭐⭐⭐⭐
  - 简介：完美展示了东方水彩的留白和淡雅特点
  - 关键词：留白、淡雅、晕染
```
