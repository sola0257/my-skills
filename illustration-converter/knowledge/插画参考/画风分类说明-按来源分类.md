## 01-现代水彩人物插画

**图片数量**：12张（IMG_5266-5277）

**风格特征**：
- 水彩画技法 + 卡通人物元素
- 色彩柔和，晕染自然
- 常见元素：花卉 + 可爱的卡通小女孩
- 人物形象：简洁、可爱、童趣
- 构图：人物与植物和谐融合
- 笔触：轻盈、透明感强

**视觉效果**：
- 清新、治愈、温馨
- 适合年轻女性受众
- 具有故事性和情感表达

**适用场景**：
- 儿童内容
- 可爱风格内容
- 轻松活泼的主题
- 年轻化、少女感内容
- 治愈系、生活美学

**与 illustration-converter skill 的对应关系**：
- 可对应：清新水彩（西方美感）
- 特点：增加了人物元素，更具故事性

## 02-中国画混合风格

**图片数量**：13张（IMG_5249-5253, 5257-5259, 5261-5265）

**风格特征**：
- 融合了多种中国画技法
- 包含：传统水墨画、现代水墨、装饰工笔画等
- 常见元素：紫色花树、亭子、竹篱笆、荷花、蜻蜓、树叶光影
- 色彩：从淡雅水墨到鲜艳工笔都有
- 笔触：流畅、写意与工整并存
- 构图：留白丰富，意境深远

**视觉效果**：
- 诗意、优雅、东方美感
- 传统与现代结合
- 多样化的表现形式

**适用场景**：
- 中式风格内容
- 传统文化主题
- 古典园林场景
- 诗意氛围营造
- 自然风光主题
- 清新文艺风格

**与 illustration-converter skill 的对应关系**：
- 可对应：水墨国画（东方美感）
- 可对应：清新水彩（东方美感）
- 特点：风格多样，涵盖了多种中国画表现形式

## 03-工笔花鸟画

**图片数量**：3张（IMG_5254-5256）

**风格特征**：
- 传统工笔画技法
- 色彩鲜艳，细节精致
- 常见元素：小鸟（蜂鸟、山雀等）+ 花朵（波斯菊、罂粟等）
- 线条细腻，勾勒清晰
- 色彩饱和度高，对比强烈
- 有蓝天背景，营造空间感

**视觉效果**：
- 精致、华丽、生动
- 具有装饰性和观赏性
- 色彩明快，视觉冲击力强

**适用场景**：
- 花鸟主题内容
- 传统节日装饰
- 精致细腻的画面需求
- 色彩丰富的场景
- 高品质插画需求

**与 illustration-converter skill 的对应关系**：
- **目前 skill 中没有对应的风格**
- 建议：可以作为新增画风"工笔花鸟画"的参考案例
- 特点：色彩鲜艳、细节丰富、装饰性强

## 04-现代装饰工笔画

- 现代与传统的完美融合

**适用场景**：

- 梦幻主题内容
- 奢华感营造
- 艺术性强的内容

**与 illustration-converter skill 的对应关系**：
