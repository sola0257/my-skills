# 全平台内容生成通用规范 v1.0

> **适用范围**：小红书、微信公众号、视频脚本、所有内容生成技能

---

## 🚫 核心禁令（所有平台必须遵守）

### 1. 禁止设定具体人设 ⚠️

**这是最重要的规范，适用于所有平台的内容生成。**

#### 为什么要避免具体人设？

1. **人设偏差风险**：臆造的具体经历可能与账号实际情况不符
2. **风格不一致**：不同内容的人设可能相互矛盾
3. **真实性问题**：虚构的具体经历容易被识破
4. **限制受众**：过于具体的身份会限制内容的普适性

#### 禁止的表述

**❌ 具体经历：**
- "去年春节，我..."
- "我第一次养花的时候..."
- "我当时特别着急..."

**❌ 具体身份：**
- "作为一个租房党..."
- "我们这些北漂..."
- "像我这样的上班族..."

**❌ 具体地域（北京除外）：**
- "我在上海租房时..."
- "我们南方人..."
- "在广州工作的朋友..."

**⚠️ 地域例外：可以使用"北京"**
- ✅ "北京的冬天比较干燥"（客观描述）
- ✅ "北京的朋友可能会遇到..."（泛化表述）
- ❌ "我在北京租房时..."（仍然禁止，因为是具体人设）

**❌ 具体时间线：**
- "我养花3年了..."
- "我从2020年开始..."
- "我之前在XX公司..."

#### 推荐的表述

**✅ 泛化主体：**
- "很多人会遇到..."
- "新手常见的问题是..."
- "大家可能会发现..."

**✅ 客观描述：**
- "这是一个常见的误区..."
- "根据经验来看..."
- "实际情况是..."

**✅ 通用场景：**
- "对于居住空间有限的朋友..."
- "上班族常见的困扰是..."
- "养花爱好者经常会..."

#### 特殊情况处理

**Q: 如果用户明确要求使用第一人称怎么办？**
A: 可以使用"我"，但避免具体化。例如：
- ✅ "我发现这个方法很有效"（泛化）
- ❌ "我去年在北京租房时发现..."（过度具体）

**Q: 故事叙事体视频怎么办？**
A: 泛化故事主角，用"很多人"代替"我"：
- ✅ "很多人第一次养绿萝，都会..."
- ❌ "我第一次养绿萝，也是..."

**Q: 如果内容需要真实感怎么办？**
A: 用细节和场景营造真实感，而不是用具体人设：
- ✅ "清晨的阳光洒在叶片上，水珠晶莹剔透"（场景细节）
- ❌ "我每天早上7点起床给它浇水"（具体人设）

## 📝 各平台特殊规范

### 小红书
- **正文格式**：禁止Markdown格式符号，使用纯文本+emoji
- **标签结构**：长尾关键词标签（3-5个）+ 其他标签（5-7个）≤10个
- **语气**：亲切口语化，但避免过度个人化

### 微信公众号
- **正文格式**：可以使用Markdown格式（会自动转换为HTML）
- **语气**：专业但亲切，娓娓道来
- **摘要**：必须生成，≤25字

### 视频脚本
- **口播**：口语化，但避免"我当时"、"我发现"等个人化表述
- **画面**：聚焦场景和动作，避免涉及博主具体身份
- **故事体**：泛化主角，用"很多人"代替"我"

## ✅ 自检清单

在生成内容前，检查以下项目：

- [ ] 是否避免了"我去年..."、"我当时..."等具体经历？
- [ ] 是否避免了"作为XX"、"像我这样的XX"等具体身份？
- [ ] 是否避免了"我在XX地方"等具体地域（北京除外）？
- [ ] 是否使用了"很多人"、"新手"、"大家"等泛化表述？
- [ ] 内容是否具有普适性，不局限于特定人群？
- [ ] 语气是否真诚自然，但不过度个人化？

## 📊 对比示例

### 示例1：开头

**❌ 错误（具体人设）：**
```
去年春节，我家花园被亲戚吐槽"连个花都养不活"。
我当时特别尴尬，下定决心要改造一下。
作为一个租房党，我预算有限，但还是想让花园好看一点。
```

**✅ 正确（泛化表述）：**
```
春节快到了，你家花园还是老样子？
很多人会担心，亲戚来串门时花园不够体面。
其实不用大动干戈，简单改造就能让花园焕然一新。
```

### 示例2：经验分享

**❌ 错误（具体经历）：**
```
我第一次养绿萝，也是各种黄叶烂根。
我每天给它浇水，生怕它渴着。
后来我才知道，我浇水太勤了。
现在我的绿萝已经爬满了半面墙。
```

**✅ 正确（客观描述）：**
```
很多人第一次养绿萝，都会遇到黄叶烂根的问题。
新手常见的误区是，每天都给它浇水。
其实绿萝喜欢"见干见湿"，不是每天都要浇。
掌握正确方法后，绿萝能长得非常茂盛。
```

### 示例3：故事叙事

**❌ 错误（过度具体）：**
```
去年春天，朋友送了我一盆绿萝。
我把它放在客厅的窗台上，每天下班回家第一件事就是看它。
有一天，我突然发现它长出了新叶，那一刻我特别感动。
```

**✅ 正确（泛化故事）：**
```
很多人收到绿萝后，都会特别用心照顾。
把它放在明亮的窗台，每天观察它的变化。
当发现它长出新叶的那一刻，会有一种特别的成就感。
```
