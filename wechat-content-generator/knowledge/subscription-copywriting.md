## 📝 标题创作规范

### 字数限制
- **严格限制**: ≤25字（含标点和emoji）
- **理想长度**: 15-20字

### 标题公式（10类）

#### 1. 数字型
```
格式: [数字] + [动作/形容词] + [核心词]
示例: "5种冬天不怕冻的绿植推荐 🌿"（15字）
```

#### 2. 痛点型
```
格式: [问题] + [解决方案]
示例: "叶子发黄？3步排查法帮你诊断"（14字）
```

#### 3. 对比型
```
格式: [A] vs [B]，[结论]
示例: "陶盆vs塑料盆，养绿萝选哪个？"（14字）
```

#### 4. 季节型
```
格式: [季节/月份] + [动作] + [对象]
示例: "一月养花预报：这些事别错过 📅"（15字）
```

#### 5. 指南型
```
格式: [对象] + [完全/新手] + 指南
示例: "新手养多肉完全指南 🌵"（11字）
```

#### 6. 误区型
```
格式: [常见误区] + [正确做法]
示例: "别再天天浇水了！绿萝怕涝不怕旱"（15字）
```

#### 7. 急救型
```
格式: [紧急情况] + 急救/补救 + [方法]
示例: "烂根急救：3天挽救你的绿植"（13字）
```

#### 8. 场景型
```
格式: [场景] + [推荐/适合的植物]
示例: "暖气房适合养什么？5种耐干植物"（15字）
```

#### 9. 揭秘型
```
格式: [现象] + 背后的真相/原因
示例: "虎皮兰不长个？可能是这3个原因"（15字）
```

#### 10. 后悔型
```
格式: 后悔没早知道 + [知识点]
示例: "后悔没早知道！换盆的最佳时间"（15字）
```

### ⚠️ 标题禁忌
- ❌ 超过25字
- ❌ 使用"震惊"、"绝了"等低质标题党词汇
- ❌ 夸大其词（"秒变养花大师"）
- ❌ emoji超过2个

## 📋 摘要创作规范

### 字数限制
- **严格限制**: ≤25字（转发时完整显示）
- **理想长度**: 15-20字

### 摘要公式

#### 1. 价值提炼型
```
格式: [核心价值] + [受众收益]
示例: "养好绿植的5个关键，新手必看"（14字）
```

#### 2. 问题导向型
```
格式: [问题] + [解决方案预告]
示例: "叶子发黄怎么办？诊断方法全解析"（15字）
```

#### 3. 结果承诺型
```
格式: [做法] + [可期待的结果]
示例: "按这个方法浇水，绿植再也不烂根"（15字）
```

### ⚠️ 摘要禁忌
- ❌ 超过25字
- ❌ 与标题高度重复
- ❌ 使用完整句子（应简洁有力）

### 结构模版

```markdown

[开篇吸引读者，带出问题或共鸣]
- 共鸣式开头："你是不是也遇到过..."
- 问题式开头："为什么我的XX总是..."
- 成果式开头："上周有花友分享了她的..."

## 一、[核心知识点A]（300-500字）
[干货内容，结构化呈现]

- 分点列出，每点2-3句话
- 加粗关键词

## 二、[核心知识点B]（300-500字）
[实操方法或案例]
- 步骤清晰，可操作
- 包含具体数据（温度、频率、比例）

![配图3]

## 三、总结（50-100字）
[一句话总结 + 引导互动]
- 重申核心观点
- 引导收藏/留言

---

[CTA区块]

[互动引导]
```

### 正文风格规范

#### 语言风格
- ✅ 口语化但不随意："咱们今天聊聊..."
- ✅ 专业但不学究：用通俗语言解释专业术语
- ✅ 亲切但不油腻：像朋友聊天，不卖弄

#### emoji使用
- 每段2-3个emoji
- 主要用于段首标记和关键词强调
- 常用emoji：🌿 🪴 ☀️ 💧 ⚠️ ✅ ❌ 💡 📌

#### 排版规范
- 段落：每段3-4行，不超过100字
- 空行：段落间空一行，增加呼吸感
- 加粗：核心观点、关键数据加粗
- 列表：3条以上信息用列表展示

## 📚 慢养四季品牌话术
