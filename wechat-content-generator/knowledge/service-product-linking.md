# 服务号商品关联指南

> 慢养四季服务号内容创作与商品关联规范

---

## 📋 服务号定位

### 账号属性
- **账号类型**: 服务号
- **核心功能**: 商品推广、小程序商城导流
- **内容方向**: 产品使用场景、好物推荐、使用教程
- **发布频率**: 每月4次（服务号限制）

### 与订阅号的协同
```
订阅号（知识输出）         服务号（商品转化）
    ↓                           ↓
 深度干货文章              场景化产品推荐
    ↓                           ↓
 建立信任和专业形象          引导购买转化
    ↓                           ↓
    └─────────── 相互引流 ───────────┘
```

## 🛍️ 商品关联策略

### 选题与商品匹配原则

#### 1. 场景匹配
从用户场景出发，推荐解决问题的商品：
```
用户场景: 新买的绿萝，不知道怎么换盆
关联内容: 《新手换盆完全指南》
关联商品: 换盆工具套装、营养土、陶粒
```

#### 2. 季节匹配
根据季节推荐应季商品：
```
季节: 春季
内容方向: 春播指南、春季换盆
关联商品: 种子套装、营养土、园艺工具
```

#### 3. 问题匹配
针对养护问题推荐对应商品：
```
问题: 叶子发黄
内容方向: 黄叶诊断与解决
关联商品: 营养液、补光灯、湿度计
```

### 商品推荐数量
- **最佳**: 2-3个商品
- **最多**: 不超过3个
- ⚠️ 商品过多会显得杂乱，降低转化


## 🔗 小程序跳转配置

### 商品图片属性
商品图片需包含小程序跳转数据：
```html
<img src="product.jpg" 
     data-miniprogram-appid="wxXXXXXXXX"
     data-miniprogram-path="pages/product/detail?id=123">
```

### 配置说明
- **appId**: 小程序的AppID
- **path**: 商品详情页路径，包含商品ID参数

### ⚠️ 重要提醒
在微信公众号后台编辑时需手动设置小程序跳转：
1. 点击商品图片
2. 选择"插入小程序"
3. 选择目标小程序
4. 设置跳转路径

## 🎨 商品配图规范

### 商品图尺寸
- **推荐比例**: 1:1 或 4:3
- **尺寸**: 800×800 或 800×600
- **格式**: PNG/JPG

### 商品图风格
- 清晰展示商品全貌
- 背景干净（白底或淡色）
- 可适当展示使用场景
- 风格与正文配图协调

## 📊 数据指标参考

### 服务号内容效果评估
| 指标 | 良好标准 | 优秀标准 |
|------|----------|----------|
| 阅读率（打开率）| >5% | >10% |
| 分享率 | >2% | >5% |
| 商品点击率 | >3% | >8% |
| 转化率（从阅读到下单）| >0.5% | >2% |

### 内容优化方向
- 阅读率低 → 优化标题
- 分享率低 → 增加干货价值
- 点击率低 → 优化商品展示位置和描述
- 转化率低 → 优化场景匹配度
