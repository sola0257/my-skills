## 📋 CTA 类型说明

### 类型1：完整版 CTA（标准）
- **适用**：常规干货文章（<2000字）
- **特点**：包含星标引导、企业微信二维码、互动引导
- **目标**：提高转化率和互动率

### 类型2：简洁版 CTA（新增）
- **适用**：深度专业文章（≥2000字）
- **特点**：只包含关于作者和免责声明
- **目标**：保持专业感，不打断阅读体验

**详细使用规范**：参考 [cta-usage-rules.md](./cta-usage-rules.md)

## 📝 使用说明

### 选择原则

**完整版 CTA（模板A-D）**：
1. **订阅号**：根据文章类型选择对应CTA模板
   - 干货文章 → 模板A或B
   - 资料引导 → 模板C
   - 季节主题 → 模板D

**简洁版 CTA（模板E）**：
1. **订阅号**：深度专业内容
   - 字数 ≥2000字
   - 科普教程类
   - 追求专业感的长文

**服务号**：
2. **服务号**：根据商品数量选择模板
   - 单品 → 模板A
   - 多品 → 模板B
   - 套装 → 模板C
   - 场景 → 模板D

**决策流程**：
```
文章字数判断
    ↓
≥2000字 → 简洁版 CTA（模板E）
<2000字 → 完整版 CTA（模板A-D）
```

### 替换变量
- `[商品名称]` → 实际商品名
- `¥XX.X` → 实际价格
- `[一句话描述]` → 商品卖点
- `product.png` → 实际商品图URL

### ⚠️ 注意事项
- 每篇文章只使用一个CTA模板
- 商品图需设置小程序跳转
- 保持CTA与正文的视觉分隔（使用分割线）
