# 微信公众号配图规范 v2.0

> **重要更新**：配图数量调整为5-6张，与文案段落匹配

---

## 📊 配图数量规范

### 总体要求
- **配图数量**：5-6张
- **封面**：1张（2.35:1横版，用于摘要）
- **正文配图**：4-5张（16:9横版）
- **匹配原则**：每个核心段落/知识点对应1张配图

### 配图分配策略

**根据内容结构动态调整：**

| 内容类型 | 段落数 | 配图数量 | 说明 |
|---------|--------|---------|------|
| 短文（3个要点） | 3-4段 | 5张 | 封面1张 + 每段1张 + 总结1张 |
| 中等文章（4-5个要点） | 5-6段 | 6张 | 封面1张 + 每段1张 |
| 长文（6+个要点） | 6+段 | 6张 | 封面1张 + 核心段落各1张 |

---

## 🎨 配图类型与分配

### 1. 封面图（1张）
- **比例**：2.35:1 横版（900×383px）
- **作用**：摘要配图，吸引点击
- **要求**：
  - 展示文章主题
  - 画面简洁大气
  - 适合横版展示

### 2. 引言配图（1张）
- **比例**：16:9 横版（900×506px）
- **作用**：引入主题，营造氛围
- **位置**：引言段落后

### 3. 核心知识点配图（3-4张）
- **比例**：16:9 横版
- **作用**：解释说明，增强理解
- **分配原则**：
  - 每个核心知识点：1张
  - 复杂知识点：可以用对比图或步骤图

**配图类型：**
- **场景图**：展示完整场景
- **对比图**：前后对比、正确vs错误
- **步骤图**：操作步骤展示
- **特写图**：细节展示

### 4. 总结配图（0-1张）
- **比例**：16:9 横版
- **作用**：总结要点，强化记忆
- **说明**：可选，如果内容较长可以添加

## 📝 配图与文案匹配规则

### 匹配原则

**每张配图必须对应文案中的具体内容：**

1. **段落匹配**：每个核心段落对应1张配图
2. **知识点匹配**：每个主要知识点对应1张配图
3. **逻辑匹配**：配图顺序与文案逻辑一致

## 🎯 配图生成策略

### 内容分析阶段

1. **统计核心段落**：计算有多少个主要知识点
2. **确定配图数**：5-6张（封面1 + 正文4-5）
3. **分配配图**：为每个核心段落分配1张配图

### Prompt生成阶段

**为每张配图生成专属Prompt：**

```
配图1（封面，2.35:1）：春节花园改造主题横幅
配图2（引言，16:9）：杂乱的花园场景
配图3（知识点1，16:9）：清理后的整洁花园
配图4（知识点2，16:9）：年宵花特写拼图
配图5（知识点3，16:9）：统一风格的花盆对比
配图6（总结，16:9）：改造完成的花园全景
```

### 风格统一原则

**同一篇文章的所有配图必须使用相同风格：**
- 选定1种风格（dreamy-photo / cozy-sketch等）
- 所有配图使用相同的Prompt风格模板
- 保持色调、光线、氛围的一致性

## ⚠️ 注意事项

1. **公众号配图不宜过多**：
   - 5-6张是合理范围
   - 过多配图会影响阅读体验
   - 保持图文平衡

2. **横版比例很重要**：
   - 封面：2.35:1（摘要配图）
   - 正文：16:9（适合手机阅读）
   - 不要使用小红书的3:4竖版

3. **配图要与文案紧密结合**：
   - 每张图对应一个核心知识点
   - 图文互补，增强理解
   - 配图位置要准确

4. **配图要符合公众号风格**：
   - 横版比例
   - 清晰、专业、有质感
   - 适合深度阅读场景
