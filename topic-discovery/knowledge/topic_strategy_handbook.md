# 选题策略核心方法论

> 来源：《绿植园艺全域运营实操手册（2026完全版）》2.5.1节

## 选题原则

**核心**：不是写"知识"，而是解决"问题"。

## 选题公式

```
用户痛点/场景 + 解决方案（商品）+ 数字/对比 = 高转化选题
```

## 选题优先级（按转化率排序）

1. **场景痛点类**（转化率最高 ⭐⭐⭐⭐⭐）
   - 示例：《客厅光线不好？这5种耐阴绿植最适合你》
   
2. **新手避坑类**（建立信任 ⭐⭐⭐⭐）
   - 示例：《养花新手必看！这3种绿植怎么养都不死》
   
3. **季节时令类**（蹭热度 ⭐⭐⭐）
   - 示例：《冬天养什么绿植？这5种耐寒又好看》
   
4. **送礼推荐类**（客单价高 ⭐⭐⭐）
   - 示例：《送长辈什么绿植好？这5种寓意好又好养》
   
5. **养护知识类**（转化率低 ⭐⭐）
   - 注意：需要强关联商品才能转化

## 场景化选品方法论

**核心原则**：不按品类选品，按场景选品。

**选品公式**：
```
场景需求 + 用户痛点 + 商品特性 = 高转化选品
```

**选品策略**：
1. **初期**：精选5-10个爆款SKU（聚焦客厅、办公桌、卧室场景）
2. **中期**：根据数据扩展到20-30个SKU
3. **长期**：建立完整的场景化商品矩阵

### 核心场景产品推荐

**客厅场景**（光线不好、空间大、需要装饰）：
- 推荐品：绿萝、龟背竹、虎皮兰、发财树
- 痛点解决：耐阴、大型、高颜值
- 价格区间：29.9-199元

**办公桌场景**（空间小、没时间、需要提神）：
- 推荐品：多肉、仙人掌、小盆栽
- 痛点解决：小型、好养、耐旱
- 价格区间：19.9-99元

**卧室场景**（影响睡眠、净化空气）：
- 推荐品：芦荟、吊兰、常春藤
- 痛点解决：夜间释放氧气、无毒
- 价格区间：25.9-149元

## 平台选题独立性

**错误做法**：所有平台用相同选题
**正确做法**：不同平台使用不同选题（基于平台特性）

### 各平台选题优先级

**小红书**：
- 优先：场景痛点类、新手避坑类、送礼推荐类
- 特点：视觉化、情绪化、种草导向

**订阅号**：
- 优先：养护教程类、季节时令类、场景痛点类
- 特点：知识化、深度、专业背书

**视频脚本**：
- 优先：养护教程类、新手避坑类、对比测评类
- 特点：动作展示、步骤清晰、画面感强

**服务号**：
- 优先：送礼推荐类、季节时令类（结合促销）
- 特点：转化导向、优惠驱动
