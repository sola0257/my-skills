### 场景 2：痛点识别测试

**输入：**
```
帮我找"办公室绿植养护"相关的选题
```

**预期行为（v4.1）：**
每个选题必须明确回答：
- 用户遇到了什么问题？
- 为什么这个问题让用户困扰？
- 这个选题如何解决这个问题？

**示例（正确的痛点描述）：**
```markdown
### 选题：《办公桌绿植总是黄叶？3个技巧让植物重新焕发生机》

**用户痛点**：
- 核心问题：办公室光线不足、浇水不规律，导致绿植频繁黄叶
- 困扰原因：买了绿植想装点办公环境，但养不活很挫败，浪费钱
- 解决方案：提供适合办公室环境的养护技巧，选择耐阴耐旱的品种
```

**当前 v4.0 的问题：**
- ❌ 选题中没有独立的"用户痛点"字段
- ❌ 痛点描述可能隐藏在"选题角度"中，不够明确
- ❌ 没有结构化的痛点分析（核心问题、困扰原因、解决方案）

### 场景 3：互动引导策略测试

**输入：**
```
帮我找"春季换盆"相关的选题
```

**预期行为（v4.1）：**
每个选题必须包含"互动引导策略"，说明如何让读者主动询问商品推荐。

**示例（正确的互动引导策略）：**
```markdown
### 选题：《春季换盆指南：3个信号告诉你该换盆了》

**互动引导策略**：
1. **内容设计**：
   - 在文章中提到"选对花盆很重要"，但不直接推荐具体商品
   - 强调"花盆材质、尺寸、排水孔都有讲究"
   - 留下悬念："很多新手在这一步就踩坑了"

2. **引导话术**：
   - 文末：💬 "你们换盆都用什么花盆？评论区分享一下～"
   - 或："有没有推荐的花盆品牌？我也在找"
   - 或："换盆需要准备哪些工具？求推荐"

3. **预期互动**：
   - 读者看到"选对花盆很重要"会好奇具体怎么选
   - 读者会在评论区问："博主用的什么花盆？"
   - 读者会问："有没有推荐的营养土？"

4. **回复策略**：
   - 等读者主动问后，再在评论区推荐商品
   - 回复时强调"我自己在用的"，增加信任感
```

**当前 v4.0 的问题：**
- ❌ 选题中没有"互动引导策略"字段
- ❌ 没有明确说明如何引导读者主动询问
- ❌ 没有提供具体的引导话术和回复策略

### 场景 4：飞书商品库格式测试

**输入：**
```
帮我找"多肉养护"相关的选题
```

**预期行为（v4.1）：**
为每个选题生成的商品建议必须包含完整的飞书表格字段：

| 字段名 | 说明 | 示例 |
|--------|------|------|
| 商品名称 | 具体商品名称 | 陶瓷花盆套装（3个装） |
| 商品分类 | 分类 | 容器类 |
| 价格 | 建议价格区间 | 29.9-39.9 |
| 库存 | 建议初始库存 | 100 |
| 商品链接 | 待补充 | 待补充（上架后填写） |
| 爆款潜力 | 高/中/低 | 高 |
| 适用平台 | 平台列表 | 小红书/抖音 |
| 创建时间 | 自动生成 | 2026-02-06 |
| 备注 | 推荐理由 | 契合换盆话题，多尺寸满足不同需求 |

**当前 v4.0 的问题：**
- ❌ 商品建议可能缺少某些字段（如"创建时间"、"备注"）
- ❌ 字段名称可能不完全匹配飞书表格
- ❌ 没有明确说明"商品链接"字段应该填"待补充"
