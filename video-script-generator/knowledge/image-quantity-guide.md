# 视频脚本配图规范 v2.0

> **重要更新**：配图数量大幅增加，用于图片+转场+运镜实现视频效果

---

## 📊 配图数量规范

### 总体要求
- **配图数量**：根据视频时长动态调整
- **基础原则**：每张配图停留3-5秒
- **转场需求**：相邻配图要考虑转场效果，给转场留足够时间
- **运镜需求**：部分配图要支持推拉摇移等运镜，不要切换太快

### 配图数量计算公式

**根据视频时长计算：**

| 视频时长 | 配图数量 | 平均每张时长 | 说明 |
|---------|---------|-------------|------|
| 30秒-1分钟 | 10-15张 | 4-5秒/张 | 短视频，节奏适中 |
| 1-2分钟 | 15-25张 | 4-5秒/张 | 中短视频，稳定节奏 |
| 2-3分钟 | 25-40张 | 4-5秒/张 | 中长视频，深度内容 |
| 3-5分钟 | 40-65张 | 4-5秒/张 | 长视频，系统讲解 |

**示例：**
- 2分30秒视频 = 150秒 ÷ 4秒/张 ≈ 38张配图

**⚠️ 重要提醒：**
- 不要1-2秒就换一张图，会让观众感觉眩晕
- 给运镜和转场留足够时间
- 宁可慢一点，也不要太快

## 🎨 配图类型与功能

### 1. 主场景图（占30-40%）
- **作用**：建立场景，营造氛围
- **特点**：
  - 画面完整，信息丰富
  - 适合推拉镜头
  - 停留时间较长（5-8秒）

**示例：**
- 花园全景
- 室内植物角落
- 阳台改造前后

### 2. 特写图（占30-40%）
- **作用**：突出细节，强调重点
- **特点**：
  - 聚焦单一主体
  - 适合缓慢切换
  - 停留时间适中（3-5秒）

**示例：**
- 植物叶片特写
- 手部操作特写
- 工具细节特写

### 3. 对比图（占10-15%）
- **作用**：展示差异，强化对比
- **特点**：
  - 左右分屏或上下分屏
  - 适合横向滑动转场
  - 停留时间适中（4-6秒）

**示例：**
- 改造前后对比
- 正确vs错误做法
- 不同植物对比

### 4. 步骤图（占10-15%）
- **作用**：展示流程，指导操作
- **特点**：
  - 连续性强
  - 适合淡入淡出转场
  - 每步3-5秒

**示例：**
- 清理步骤1、2、3
- 种植步骤1、2、3
- 养护步骤1、2、3

### 5. 过渡图（占5-10%）
- **作用**：连接段落，平滑过渡
- **特点**：
  - 画面简洁
  - 适合快速过渡
  - 停留时间稍短（3-4秒）

**示例：**
- 日历翻页
- 时钟走动
- 季节变化

## 🎬 配图与分镜匹配规则

### 匹配原则

**每个分镜通常需要1-3张配图：**

1. **短分镜（3-5秒）**：1张配图 + 运镜效果
2. **中等分镜（5-10秒）**：1-2张配图 + 转场效果
3. **长分镜（10秒+）**：2-3张配图 + 多重转场

**⚠️ 重要：不要过度切换**
- 每张配图至少停留3-5秒
- 给运镜和转场留足够时间
- 避免1-2秒就换图，会让观众眩晕

### 分镜配图分配示例

- 配图1：杂乱的花园（2秒，推镜头）

**分镜02：时间紧迫感（8秒）**
- 配图3：日历特写（4秒，静止）
- 配图4：春节标注（4秒，放大）

**分镜03：第一步-清理（15秒）**

- 配图6：清理过程（5秒，特写）
- 配图7：清理后全景（5秒，推镜头）

## 🎯 配图生成策略

### 内容分析阶段

1. **统计分镜数**：计算有多少个分镜
2. **计算视频时长**：累加所有分镜时长
3. **确定配图数**：时长÷4秒 = 配图数量
4. **分配配图**：为每个分镜分配2-5张配图

### 配图分配规则

**根据分镜时长分配配图数量：**

| 分镜时长 | 配图数量 | 说明 |
|---------|---------|------|
| 3-5秒 | 1张 | 短分镜，单图+运镜 |
| 5-10秒 | 1-2张 | 中等分镜，适度变化 |
| 10-15秒 | 2-3张 | 长分镜，丰富画面 |
| 15秒+ | 3-4张 | 超长分镜，多重变化 |

**⚠️ 注意：每张配图至少停留3-5秒**

### Prompt生成阶段

**为每张配图生成专属Prompt：**

```
分镜01（5秒，2张配图）：
- 配图1：杂乱的花园全景，枯叶散落
- 配图2：改造后的花园全景，整洁喜庆

分镜02（8秒，2张配图）：
- 配图3：2026年1月日历特写
- 配图4：2月17日春节标注

分镜03（15秒，3张配图）：
- 配图5：清理前的杂乱花园
- 配图6：清理过程特写
- 配图7：清理后的整洁花园

...
```

## 🎬 转场与运镜建议

### 常用转场效果

1. **淡入淡出**：适合场景切换
2. **横向滑动**：适合对比图
3. **推拉镜头**：适合全景到特写
4. **闪切**：适合快节奏切换
5. **叠化**：适合时间流逝

### 常用运镜效果

1. **推镜头**：从远到近，聚焦主体
2. **拉镜头**：从近到远，展示全景
3. **摇镜头**：左右摇动，展示空间
4. **移镜头**：平移跟随，展示动作
5. **缩放**：放大缩小，强调细节

### 配图设计建议

**为了支持运镜效果，配图要：**
- 画面留白充足（支持推拉镜头）
- 主体位置多样（支持摇移镜头）
- 细节丰富（支持缩放效果）
- 构图稳定（支持转场效果）

## ⚠️ 注意事项

1. **配图数量要合理**：
   - 按照3-5秒/张的节奏
   - 不要过多导致切换太快
   - 不要过少导致画面单调

2. **配图要考虑转场**：
   - 相邻配图要有连贯性
   - 避免画面跳跃过大
   - 给转场效果留足够时间（至少0.5-1秒）

3. **配图要支持运镜**：
   - 画面要有足够的空间
   - 主体位置要合理
   - 细节要丰富
   - 每张图至少停留3-5秒才能完成运镜

4. **配图要与口播匹配**：
   - 画面要与口播内容一致
   - 时长要与口播节奏匹配
   - 重点画面要与重点口播对应

5. **风格要统一**：
   - 同一视频所有配图风格一致
   - 色调、光线、氛围保持统一
   - 避免风格混乱

6. **避免过快切换**：
   - ❌ 不要1-2秒就换一张图
   - ✅ 每张图至少停留3-5秒
   - ⚠️ 过快切换会让观众感觉眩晕
