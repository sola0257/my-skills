# 视频脚本生成规范 v2.0

> **重要更新**：避免具体人设、口播语气规范

---

## 🚫 核心禁令（必须遵守）

### 1. 禁止设定具体人设

**❌ 错误示例：**
- "我去年养花的时候..."
- "我第一次养绿萝，也是各种黄叶烂根..."
- "作为一个北漂，我..."
- "我在上海租房的时候发现..."（北京除外）

**✅ 正确示例：**
- "很多人第一次养绿萝，都会遇到黄叶烂根..."
- "新手常见的问题是..."
- "对于居住空间有限的朋友..."
- "大家可能会发现..."

**原则：**
- 使用中性化、通用化的表述
- 避免臆造博主的具体经历、身份
- 地域可以使用"北京"，其他地域要排除
- 用"很多人"、"大家"、"新手"等泛指词汇
- 用客观描述代替主观经历
- 即使是"故事叙事体"，也要用泛化的故事，不要具体到个人

## 🎬 分镜脚本规范

### 画面描述（避免具体人设）

**❌ 错误描述：**
- "博主站在自家阳台，回忆去年的经历"
- "博主展示自己租房时的小花园"

**✅ 正确描述：**
- "阳台花园的全景展示"
- "手部特写：正在整理植物"
- "背影：站在窗边看向花园"

**原则：**
- 画面描述聚焦于场景、动作、植物
- 避免涉及博主的具体身份背景
- 如需人物出镜，用"手部特写"、"背影"等方式

## 📝 逐字稿规范

### 口播语气（避免具体人设）

**❌ 错误口播：**
```
我第一次养绿萝，也是各种黄叶烂根。
我当时特别着急，每天都给它浇水。
后来我才知道，我浇水太勤了。
```

**✅ 正确口播：**
```
很多人第一次养绿萝，都会遇到黄叶烂根的问题。
新手常见的误区是，每天都给它浇水。
其实绿萝喜欢"见干见湿"，不是每天都要浇。
```

**原则：**
- 用"很多人"、"新手"、"大家"代替"我"
- 用"常见的问题是"代替"我当时"
- 用"可能会发现"代替"我发现"
- 保持口语化，但避免过度个人化

## 🎯 开头规范（3秒钩子）

### ❌ 错误开头
- "我去年春节的时候，花园被吐槽了..."
- "作为一个养花3年的人，我想说..."
- "我在北京租房时发现..."

### ✅ 正确开头
- "春节快到了，你家花园还是老样子？"
- "你是不是也觉得绿萝特别好养？"
- "很多人在这个问题上卡很久..."

**开头公式：**
1. **反问式**：你是不是也...
2. **痛点式**：很多人都遇到...
3. **场景式**：春节快到了...
4. **好奇式**：你知道吗...

## 💬 语气与调性

### 推荐语气
- 亲切自然，像跟朋友聊天
- 真诚但不煽情
- 专业但不说教
- 有共鸣但不过度代入

### 称呼规范
- ✅ 姐妹们、朋友们、大家
- ✅ 新手、爱养花的朋友
- ✅ 你、你们
- ❌ 我们租房党、我们北漂
- ❌ 像我这样的...

## 🎭 故事叙事体的特殊处理

即使是"故事叙事体"视频，也要避免具体人设：

**❌ 错误（过度具体）：**
```
去年春天，朋友送了我一盆绿萝。
我每天给它浇水，生怕它渴着。
结果两周后，叶子开始发黄，根部也烂了。
我以为它要死了。
```

**✅ 正确（泛化故事）：**
```
很多人收到绿萝后，都会特别用心照顾。
每天浇水，生怕它渴着。
但往往两周后，叶子就开始发黄，根部也烂了。
这时候大家都会担心，是不是养不活了。
```

**原则：**
- 用"很多人"代替"我"
- 用"往往"、"常常"代替"当时"
- 保持故事性，但泛化主角
- 让观众能代入，而不是只听博主的个人经历
