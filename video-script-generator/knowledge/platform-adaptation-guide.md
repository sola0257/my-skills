# 视频平台适配指南 v1.0

> 本文档定义四大平台（小红书/视频号/快手/抖音）的内容偏好和适配策略。

---

## 一、平台概览

| 平台 | 核心用户 | 内容调性 | 推荐时长 | 黄金节奏 |
|------|----------|----------|----------|----------|
| 🔴 小红书 | 年轻女性/都市白领 | 精致、种草、姐妹感 | 1-3分钟 | 紧凑、信息密集 |
| 🟣 视频号 | 30+用户/熟人社交 | 品质、调性、有深度 | 1-5分钟 | 中等、娓娓道来 |
| 🟡 快手 | 下沉市场/真实生活 | 接地气、实用、真诚 | 30秒-3分钟 | 直接、干货优先 |
| 🔵 抖音 | 年轻人/娱乐向 | 潮流、快节奏、网感 | 15秒-1分钟 | 快、钩子强 |

## 二、小红书 🔴

### 内容偏好
- ✅ 有干货、能学到东西
- ✅ 视觉美感、有质感
- ✅ 姐妹分享感、真诚
- ✅ 适当的生活方式展示
- ❌ 过于专业/硬核
- ❌ 粗糙的画面

### 开头钩子模板
```
诶，你是不是也[痛点]？
姐妹们，这个真的绝了！
后悔没早知道！[好处]
天哪，原来[发现]！
```

### 话术风格
- 第二人称为主：你、姐妹们
- 适度夸张：绝了、太香了、疯狂心动
- 情绪表达丰富
- 口语化+网感

### 字幕特点
- 关键词放大加粗
- 金句单独成屏
- emoji适当点缀
- 节奏感强（每3-5秒一个信息点）

### 结尾CTA
```
有用的话记得点赞收藏呀～
关注我，下期更精彩！
评论区告诉我你的[问题]
需要的姐妹评论区扣1
```

## 三、视频号 🟣

### 内容偏好
- ✅ 有深度、有内涵
- ✅ 品质感、调性高
- ✅ 图文公众号的视频延伸
- ✅ 适合转发到朋友圈
- ❌ 过于娱乐化
- ❌ 低质量内容

### 开头钩子模板
```
今天想和大家聊聊[话题]
很多朋友都问我[问题]
最近发现一个[洞察]
分享一个让我[感受]的事
```

### 话术风格
- 温和有质感
- 娓娓道来，不急不躁
- 知识性与情感性平衡
- 像跟老朋友分享

### 字幕特点
- 简洁大方
- 不过度设计
- 关键词突出即可
- 配合公众号风格

### 结尾CTA
```
关注我的公众号，有更详细的分享
点赞转发，让更多人看到
有任何问题，评论区交流
喜欢的话，帮我点个在看
```

### 私域联动
- 可提及公众号名称
- 引导添加企业微信
- 社群预热提示

## 四、快手 🟡

### 内容偏好
- ✅ 真实、接地气
- ✅ 实用性强、能直接用
- ✅ 体验感、过程感
- ✅ 老铁感、信任感
- ❌ 太精致/装腔作势
- ❌ 纯理论无实操

### 开头钩子模板
```
家人们，今天教你们一招
这个方法简单实用，学会了[好处]
很多老铁都问我[问题]
今天手把手教你[技能]
```

### 话术风格
- 朴素直接
- 老铁、家人们
- 体验式分享
- 不绕弯子

### 字幕特点
- 大字、清晰
- 重点用红色或黄色
- 步骤标号清晰
- 土味但有效

### 结尾CTA
```
觉得有用给个双击
有问题评论区问我
关注我学更多实用技巧
点点关注不迷路
```

## 五、抖音 🔵

### 内容偏好
- ✅ 快节奏、强钩子
- ✅ 网感、潮流
- ✅ 娱乐性与知识性结合
- ✅ 热点结合
- ❌ 节奏太慢
- ❌ 开头平淡

### 开头钩子模板（3秒黄金法则）
```
停！你[错误行为]了吗？
1个动作，搞定[问题]
99%的人都不知道的[秘密]
千万别这样[错误]！
3秒学会[技能]
```

### 话术风格
- 快节奏，句子短
- 网络流行语适度
- 情绪饱满
- 有梗有料

### 字幕特点
- 大号字体
- 强调色（红黄）
- 关键数字放大
- 节奏感超强（2-3秒一换）

### 结尾CTA
```
点赞收藏，下期更绝！
关注我，涨知识不迷路
评论区告诉我你的情况
双击有惊喜
```

### 热点结合
- 关注当前热点话题
- BGM选择流行音乐
- 话题标签跟热点

## 六、通用版本指南

当未指定平台时，生成通用版本：

### 取中间值
- 节奏：中等偏紧凑
- 时长：1-2分钟
- 话术：亲切但不过度口语

### 开场模板
```
你可能不知道，[发现/秘密]
很多人都问我[问题]
今天来聊聊[话题]
```

### 结尾模板
```
喜欢的话记得点赞关注
有问题评论区见
下期再聊，拜拜～
```

## 七、同一选题的多平台改编

### 改编三要素
1. **钩子调整**：按平台钩子风格重写
2. **节奏调整**：抖音快、视频号慢
3. **话术微调**：老铁/姐妹/朋友

### 改编示例

**选题：绿萝浇水技巧**

| 平台 | 开头钩子 |
|------|----------|
| 小红书 | 姐妹们！绿萝总是黄叶？可能你浇水浇错了！ |
| 视频号 | 今天想和大家分享一个养绿萝的小技巧 |
| 快手 | 家人们，绿萝养不活？跟我学这招准没错 |
| 抖音 | 停！绿萝这样浇水必死！ |
