## 技术要点

### 飞书 API 使用经验

1. **字段名称 vs 字段ID**
   - ✅ 添加记录时使用**字段名称**
   - ❌ 不要使用字段ID
   - 示例: `{"标题": "测试"}` ✅
   - 错误: `{"fld43IC5iv": "测试"}` ❌

2. **日期字段格式**
   - 飞书 API 的日期字段（type 5）需要 Unix 时间戳（毫秒）
   - Python 转换: `int(datetime.timestamp() * 1000)`
   - 示例: `2026-02-05` → `1770220800000`

3. **API 端点**
   - 重命名应用: `PUT /bitable/v1/apps/{app_token}`
   - 重命名表格: `PATCH /bitable/v1/apps/{app_token}/tables/{table_id}`
   - 添加记录: `POST /bitable/v1/apps/{app_token}/tables/{table_id}/records`
