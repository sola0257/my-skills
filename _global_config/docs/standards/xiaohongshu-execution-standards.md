## 🎯 强制执行标准

### 1. Emoji 使用规范

**要求**：小红书内容必须包含 emoji

**使用场景**：
- 标题中：1-2个相关emoji
- 正文中：每2-3段使用1个emoji
- 情绪表达：😭😵🤦‍♀️💧等
- 列表标记：1️⃣2️⃣3️⃣ 或 ✅❌
- 结尾引导：📋💡🌱等

**示例**：
```
❌ 没有emoji：
最近好多朋友问我，为啥同样养绿萝...

✅ 有emoji：
最近好多朋友问我，为啥同样养绿萝，我说7天浇一次，她养死了🤦‍♀️
```

**检查点**：
- [ ] 标题有emoji
- [ ] 正文每2-3段有emoji
- [ ] 列表使用emoji标记
- [ ] 结尾引导有emoji

### 2. 人说话的方式（强制）

**核心原则**：模仿真人说话，不是机器写作

#### 2.1 短句为主

**❌ AI写法**（长句、复杂结构）：
```
其实这不是真的焦虑，而是植物在感知季节变化时的生理调整，1月底到2月初正值冬春交替期，虽然气温还很低，但光照时长开始变化，植物的生物钟已经开始"苏醒"。
```

**✅ 人说话**（短句、简单结构）：
```
其实这不是真的焦虑。

是植物在感知季节变化。

1月底到2月初，冬春交替。

气温还很低，但光照时长开始变化了。

植物的生物钟，已经开始"苏醒"。
```

#### 2.2 口语化表达

**推荐用词**：
- "好多朋友问我" > "很多人询问"
- "养死了" > "养护失败"
- "干得要命" > "非常干燥"
- "后来才发现" > "经过分析发现"
- "对号入座" > "根据自身情况选择"

**口语化标记**：
- 啊、呢、吧、哦
- 例：环境完全不一样啊！
- 例：表面干了，里面可能还湿着呢。

#### 2.3 避免的AI痕迹

**❌ 禁止使用**：
- 过度完美的逻辑结构
- "首先...其次...最后..."
- "综上所述"
- "总而言之"
- 过多的因果关系词（因此、所以、由于）

**✅ 改用**：
- 自然的思维跳跃
- "先说个..."
- "对了..."
- "还有一点..."
- 简单的连接（然后、后来、结果）

#### 2.4 真实感细节

**必须包含**：
- 个人经历："我刚开始养植物的时候..."
- 踩坑经历："结果还是养死了好几盆😭"
- 朋友案例："我朋友在广州，她说..."
- 真实对话："她问我为啥..."

### 3. 日期准确性（强制）

**要求**：所有日期和星期必须准确对应

**检查方法**：
```python
from datetime import datetime

# 获取当前日期
today = datetime.now()
print(f"今天是 {today.strftime('%Y年%m月%d日')} {today.strftime('%A')}")

# 检查目标日期
target_date = datetime(2026, 1, 31)
print(f"目标日期是 {target_date.strftime('%Y年%m月%d日')} {target_date.strftime('%A')}")
```

**常见错误**：
- ❌ "1月31日（周五）" - 需要验证
- ❌ "明天是周六" - 需要基于当前日期计算

**正确做法**：
- ✅ 使用系统时间验证
- ✅ 明确标注"今天是X月X日周X"

### 4. 严格的价值自检（强制）

**标准**：必须满足以下至少2项

#### 4.1 省时间

**判断标准**：
- ✅ 用户不用花时间查资料
- ✅ 用户能快速判断/操作
- ✅ 提供现成的对照表/清单

**示例**：
- ✅ "浇水频次对照表" - 直接看表，不用查
- ❌ "植物换季焦虑" - 需要自己理解和判断

#### 4.2 省钱

**判断标准**：
- ✅ 避免买错东西
- ✅ 避免植物养死重新买
- ✅ 避免买不必要的工具

**示例**：
- ✅ "不用买测湿仪，3个土办法" - 省钱
- ❌ "植物美学与空间设计" - 不省钱

#### 4.3 省力

**判断标准**：
- ✅ 减少试错次数
- ✅ 简化操作步骤
- ✅ 提供简单判断方法

**示例**：
- ✅ "手指戳土2-3厘米判断" - 简单
- ❌ "从植物生理学角度解析" - 复杂

#### 4.4 直接解决问题

**判断标准**：
- ✅ 用户看完能立即行动
- ✅ 有明确的可执行方案
- ✅ 解决用户正在焦虑的问题

**示例**：
- ✅ "北方vs南方浇水频次对照" - 立即可用
- ❌ "植物的时间感" - 无法立即行动

**价值自检输出格式**：

```markdown
## 价值自检

1. 省时间：✅/❌ [具体说明]
2. 省钱：✅/❌ [具体说明]
3. 省力：✅/❌ [具体说明]
4. 直接解决：✅/❌ [具体说明]

满足项数：X/4
判断：通过/不通过（至少2项）
```

### 5. 植物多样性（强制）

**问题**：总是用绿萝、发财树等高频植物做例子

**要求**：
- 每篇内容至少涉及3种不同植物
- 避免连续使用同一植物做主要案例
- 优先使用用户问题中提到但不常见的植物

**植物使用频率监控**：

最近7天使用次数：
- 绿萝：X次
- 发财树：X次
- 琴叶榕：X次
- ...

**规则**：
- 同一植物，7天内最多作为主要案例1次
- 作为列举案例，可以多次出现但不能是唯一案例

**多样化策略**：

**常见植物**（可以用，但要控制频率）：
- 绿萝、吊兰、发财树、橡皮树、琴叶榕

**应该多用的植物**：
- 观叶类：龟背竹、春羽、竹芋类、蕨类
- 观花类：蝴蝶兰、君子兰、茉莉、栀子花
- 多肉类：生石花、肉锥花、景天科各种
- 小众类：空气凤梨、鹿角蕨、苔藓

### 6. 内容结构规范

#### 6.1 开头（必须有钩子）

**要求**：前3句话必须吸引注意力

**方法**：
- 提出问题："最近好多朋友问我..."
- 制造反差："同样养绿萝，我说7天浇一次，她养死了"
- 个人经历："我刚开始养植物的时候..."

#### 6.2 正文（必须有结构）

**要求**：
- 分段清晰，每段3-5句话
- 使用小标题分隔
- 列表使用emoji标记

**结构示例**：
```
开头钩子（3句话）

先说个判断方法
[具体方法]

北方的情况
[具体说明]

南方的情况
[具体说明]

对照表来了
[表格或列表]

几个特殊情况
[补充说明]

最重要的一点
[总结 + 个人经验]
```

#### 6.3 结尾（必须有互动）

**要求**：
- 引导评论区互动
- 提供资料领取钩子
- 使用emoji

**示例**：
```
大家可以在评论区说说，你在哪个城市？你家植物多久浇一次？

我们互相参考一下～

需要更详细的《不同城市浇水指南》的朋友，欢迎关注我的后续更新📋
```

#### 6.4 互动禁忌（严重警告）

**❌ 严禁使用以下诱导话术（会导致违规/限流）：**
- "评论打'要指南'领取"
- "扣1发资料"
- "把XX打在公屏上"
- "私信我领教程"

**✅ 推荐使用：**
- "欢迎在评论区分享你的看法"
- "关注我，持续分享养花干货"
- "有问题欢迎提问，我尽量回复"

## 📋 执行检查清单

**每次生成内容前，必须确认**：

### 内容规划
- [ ] 已读取最新用户问题监测
- [ ] 已查询营销日历
- [ ] 已检查已发布内容
- [ ] 已进行严格价值自检（至少2/4项）
- [ ] 已确认植物多样性（不重复高频植物）

### 内容生成
- [ ] 使用短句（平均每句<20字）
- [ ] 使用口语化表达
- [ ] 包含个人经历/踩坑故事
- [ ] 避免AI痕迹（完美逻辑、因果词）
- [ ] 日期和星期准确对应

### 格式检查
- [ ] 标题有emoji
- [ ] 正文每2-3段有emoji
- [ ] 列表使用emoji标记
- [ ] 结尾有互动引导
- [ ] 结尾有emoji

### 发布版格式规范（_发布版.txt）

**字数限制**：
- ✅ 标题：不超过20个字
- ✅ 全文：不超过1000个字（包括话题标签）

**段落格式**：
- ❌ 错误：一句话就空一行
  ```
  最近好多朋友问我。

  立春快到了。

  能不能给植物换盆。
  ```

- ✅ 正确：按段落分，段落之间空一行
  ```
  最近好多朋友问我，立春快到了，能不能给植物换盆、施肥🌱

  我自己养了十几年植物，也踩过不少坑。今天客观分享一下我的经验。

  其实我刚开始养植物的时候，也是一到春天就着急换盆施肥。结果有几盆植物反而出问题了😭
  ```

**段落定义**：
- 一个完整的意思/观点 = 一个段落
- 段落内可以有多句话
- 段落之间空一行

**检查点**：
- [ ] 标题≤20字
- [ ] 全文≤1000字（含标签）
- [ ] 段落格式正确（不是一句一行）
- [ ] 段落之间有空行

### 最终检查
- [ ] 读一遍，听起来像人说话吗？
- [ ] 有没有过长的句子？
- [ ] 有没有过于完美的逻辑结构？
- [ ] 植物案例是否多样？
