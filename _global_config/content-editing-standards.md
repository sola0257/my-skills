# 内容编辑场景统一规范 v1.0

**创建日期**：2026-01-30
**适用范围**：所有内容生成类 Skills（wechat、xiaohongshu、video）

---

## 🎯 核心原则

**编辑场景虽然是简化流程，但必须遵循所有 Skill 规范**

- ✅ 简化的部分：苏格拉底式提问、去重检查、选题确定
- ❌ 不能简化的部分：质量检查、配图规范、违禁词检查、账号阶段策略

---

## 🔍 场景识别

### 触发词

**编辑场景触发词**：
- "修改"、"调整"、"改一下"
- "优化"、"重写"、"完善"
- "更新"、"润色"、"改进"

**新建场景触发词**：
- "生成"、"创建"、"写一篇"
- "帮我生成XX"、"创建一篇XX"

---

## 📝 编辑场景标准流程

### Step E0: 场景识别和内容定位 [强制]

**操作**：
1. 识别用户意图（修改/优化/重写）
2. 确定要编辑的内容位置
3. 读取已有内容

**询问**：
- "请提供要编辑的内容路径或日期"
- 或"请描述要编辑的内容"

---

### Step E1: 读取已有内容 [强制]

**操作**：
1. 读取内容文档
2. 读取配图文件
3. 读取元数据（data.json）
4. 分析当前内容的：
   - 格式类型
   - 平台类型
   - 账号阶段
   - 商品植入情况

---

### Step E2: 询问修改需求 [强制]

**使用 AskUserQuestion 工具询问**：

```
问题1：需要修改哪些部分？（可多选）
选项：
- 标题
- 正文内容
- 配图
- CTA/引导语
- 全部重写

问题2：修改的原因或目标是什么？
选项：
- 优化质量（提升可读性）
- 调整商品植入度
- 修正错误信息
- 适配不同平台
- 其他（请描述）
```

---

### Step E3: 执行修改 [必需，静默执行]

**根据修改需求执行对应操作**，遵循对应平台的所有规范。

---

### Step E4: 质量检查 [强制，静默执行]

**⚠️ 这是最关键的步骤，不能因为是"修改"就跳过**

#### 必须执行的检查项

1. **违禁词检查**
   - 调用 `compliance-checker`
   - 确保无违禁词、夸张用语、虚假描述

2. **配图规范检查**
   - 小红书：12-15张（3:4竖版，1080×1440）
   - 微信长文：1封面（2.35:1）+ 3正文（16:9）
   - 微信图文：1封面（3:4）+ 5正文（3:4）
   - 视频：根据平台选择尺寸
   - 确保尺寸、数量、风格符合要求

3. **内容质量检查**
   - 标题符合平台要求
   - 正文结构清晰
   - CTA 完整
   - 符合平台调性

4. **账号阶段策略检查**
   - 商品植入度符合账号阶段
   - 起号期（0-500粉）：仅作案例提及，禁止价格
   - 成长期（500-2000粉）：软植入，可提价格
   - 成熟期（2000+粉）：可挂链接，可说价格

5. **信任型内容原则检查**
   - 客观真实
   - 避免情绪化标题
   - 避免夸张绝对化
   - 真实体验分享

**如果发现问题**：
- 自动修正（如违禁词替换）
- 或提示用户需要调整

---

### Step E5: 保存文件 [必需，静默执行]

**操作**：
1. 保存修改后的文档
2. 保存新生成的配图（如果有）
3. 更新元数据
4. 标注"已修改"和修改时间

---

### Step E6: 询问后续操作 [必需]

**使用 AskUserQuestion 工具询问**：

```
问题：是否需要重新推送/发布？
选项：
- 是（立即推送）
- 否（仅保存本地）
- 稍后手动推送
```

- [ ] 已识别为编辑场景
- [ ] 已读取已有内容
- [ ] 已询问修改需求
- [ ] 已执行修改

- [ ] 已保存文件
- [ ] 已询问后续操作

## 🚫 常见错误

### 错误1：跳过质量检查

❌ **错误做法**：
```
用户：修改一下标题
AI：好的，已修改标题为"XXX"
（没有进行违禁词检查、没有检查标题长度）
```

✅ **正确做法**：
```
用户：修改一下标题
AI：好的，已修改标题为"XXX"
    - 违禁词检查：通过
    - 标题长度：23字（符合要求）
    - 信任型原则：符合
```

### 错误2：降低配图质量

❌ **错误做法**：
```
用户：优化一下正文
AI：已优化正文，配图保持不变
（没有检查配图是否符合规范）
```

✅ **正确做法**：
```
用户：优化一下正文
AI：已优化正文
    - 配图检查：发现配图数量不足（只有10张，应该12-15张）
    - 已重新生成配图，现在有12张
```

### 错误3：忽略账号阶段

❌ **错误做法**：
```
用户：重写一下，增加商品推荐
AI：已重写，增加了3个商品推荐，包含价格和购买链接
（没有检查账号阶段，起号期不应该有价格和链接）
```

✅ **正确做法**：
```
用户：重写一下，增加商品推荐
AI：检测到当前账号处于起号期（0粉丝）
    - 已重写，增加了商品案例提及（如："我自己用的是XX"）
    - 未包含价格（符合起号期策略）
    - 未包含购买链接（符合起号期策略）
```

## 📚 平台特定规范

### 小红书编辑场景

**必须遵循**：
- 双标题系统（视觉标题 + 搜索标题）
- 12-15张配图（3:4竖版）
- 封面设计规范（白色粗体+黑边）
- 信任型内容策略

### 微信编辑场景

**必须遵循**：
- 长文/图文格式规范
- 配图尺寸和数量
- 订阅号 vs 服务号策略
- CTA 规范

### 视频编辑场景

**必须遵循**：
- 分镜脚本结构
- 4平台封面规范
- 口播文案质量
- 视频类型策略
