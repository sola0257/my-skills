# 通用图像提示词优化模板

**来源**: prompt-optimizer 项目
**版本**: v1.3.0
**适用场景**: 优化图像生成提示词，适用于小红书配图、微信配图、视频封面等

## 核心优化策略

### 1. 主体与动作
- 用 2-3 个精准修饰词刻画形态、表情与质感
- 加入明确动作或与道具的互动，增强叙事性

### 2. 环境与空间
- 设置可识别的环境锚点（小屋/花园小径/林间空地等）
- 明确前景/中景/背景层次，稳定空间关系

### 3. 光线与时间
- 描述光质与方向（柔和/硬朗，侧光/逆光/顶光等）
- 指明时间氛围（清晨/黄昏/夜景）

### 4. 色彩与材质
- 主色倾向与互补对比（粉彩/暖土色/冷色系）
- 材质质感与画面肌理（手绘线条/柔和笔触/纸张颗粒）

### 5. 氛围与风格
- 用抽象风格词表达统一审美（童话/故事书/田园温馨）

### 6. 构图与视角
- 说明画幅（正方形构图/方幅画面）
- 镜头距离（特写/半身/大全景）
- 视角（平视/俯视/仰视）

## 模板内容

```
请将以下描述优化为通用的自然语言图像提示词：

说明：
- 仅使用自然语言；不包含参数、权重或负面清单
- 输出 3-6 个结构化的句子，每句专注 1 个核心维度
- 每个关键名词配 2-3 个精准修饰词（如"柔和、漫射的晨光"）
- 句式参考：主体+动作+环境锚点 → 光照+时间+配色 → 氛围+风格 → （可选）材质纹理/构图视角

原始描述：
[在这里插入原始图像描述]

请输出优化后的提示词：
```

## 使用示例

### 输入
```
一只熊猫在竹林里
```

### 输出
```
蓬松黑白毛发的大熊猫，正坐在青翠的竹林间，周围是高耸的竹子和散落的竹叶。柔和的、漫射的晨光透过竹叶缝隙洒下，形成斑驳的光影，整体呈现清新的绿色调。画面营造出宁静、自然的氛围，带有童话故事书般的温馨感。采用正方形构图，平视角度，突出熊猫的可爱表情和周围环境的和谐。
```

## 质量保证

**使用此模板后，图像提示词质量提升体现在**：
1. 描述更详细：从简单描述扩展为多维度描述
2. 结构更清晰：主体→环境→光线→色彩→氛围→构图
3. 生成更稳定：详细的描述减少随机性，提高可控性

## 适用场景

✅ **适合使用**：
- 小红书配图（需要真实感、生活气息）
- 微信公众号配图（需要专业、清晰）
- 视频封面（需要吸引力、情绪感）

❌ **不适合使用**：
- 简单的图标、logo
- 纯文字图片
- 技术图表
