# Step 5 内容生成检查清单

> 整合8个文档的核心要点
> 生成文案前必须检查

---

## 一、平台限制（强制）

- [ ] **标题≤20字**（小红书正文标题强制限制）
- [ ] **正文≤1000字**（小红书正文强制限制）
- [ ] 视觉标题10-15字（封面显示效果最佳）

---

## 二、标题公式

### 信任型标题（成熟期账号推荐）
- [ ] 客观描述 + 价值点 + 时间/经验
- [ ] 推荐用词："客观来说"、"个人体验是"、"根据我的经验"
- [ ] 避免用词："必买!"、"绝了!"、"姐妹们冲!"

### 爆款型标题（起号期账号推荐）
- [ ] 人群+痛点+解决方案
- [ ] 数字+结果+反常识
- [ ] 价值+互动+钩子

### 双标题系统（小红书特有）
- [ ] 视觉标题（封面大字）：10-15字，情绪化，吸引点击
- [ ] 正文标题（笔记标题）：≤20字，包含SEO关键词

### SEO优化
- [ ] 包含核心关键词（植物名称、养护动作、问题描述、场景描述）
- [ ] 包含长尾关键词（人群标签、时间节点、风格标签）

---

## 三、内容结构

### 黄金模板（快速内容）
- [ ] 开头(100-150字)：抛出痛点或成果
- [ ] 中间(600-700字)：给出1、2、3个具体步骤
- [ ] 结尾(100-150字)：总结升华，引导互动

### 5步信任型框架（深度内容）
- [ ] 开头(100-150字)：建立共鸣+权威背景
- [ ] 背景(100-150字)：增加可信度
- [ ] 核心(500-600字)：价值输出
- [ ] 总结(100-150字)：客观评价
- [ ] 引导(50-100字)：引导行动

---

## 四、内容方向（必须混搭）

### 混搭玩法（实际使用）
- [ ] 颜值+求生：30%颜值 + 70%求生
- [ ] 懒人+颜值：40%懒人 + 60%颜值
- [ ] 求生+懒人：50%求生 + 50%懒人

### 禁止单独使用
- [ ] 不要只用"新手求生"
- [ ] 不要只用"懒人省心"
- [ ] 不要只用"颜值正义"

---

## 五、互动话术

### 6种话术类型
- [ ] 问题引导型："你学会了吗？评论区告诉我"
- [ ] 诊断服务型："评论区发图，我来帮你诊断"
- [ ] 预告钩子型："关注我，下期分享XX"
- [ ] 情感连接型："关注我，和我一起XX"
- [ ] 选择题式："A/B/C，你选哪个？"
- [ ] 许愿式："留下你想要的，我帮你找"

### 严禁诱导互动（平台红线）
- [ ] 禁止："评论打'要资料'"
- [ ] 禁止："扣1发教程"
- [ ] 禁止："想要链接的私信我"
- [ ] 禁止："评论区留个小心心"

### 正确引导
- [ ] "欢迎在评论区分享你的观察"
- [ ] "感兴趣的朋友可以关注我，下期详细分享"
- [ ] "关于XX还有不懂的，欢迎在评论区提问"

---

## 六、反面案例（必须避免）

### 标题错误
- [ ] 避免缺少"钩子"（平铺直述）
- [ ] 避免过于夸张（"绝了!"、"太爱了!"）
- [ ] 避免标题与内容不符
- [ ] 避免过长（>20字）或过短（<10字）

### 正文错误
- [ ] 避免过于"完美"（像说明书，缺少人情味）
- [ ] 避免内容过于基础（"要浇水、要施肥"）
- [ ] 避免内容过于宽泛（没有针对性）
- [ ] 避免字数超过1000字

### 互动错误
- [ ] 避免不够"具体"（提问太宽泛）
- [ ] 避免缺少"激励"（没有回报）
- [ ] 避免过于"营销化"（"快关注我，不关注你会后悔!"）

---

## 七、信任型内容原则（全平台适用）

### 推荐用词
- [ ] "客观来说..."、"个人体验是..."
- [ ] "根据我的经验..."、"建议大家..."
- [ ] "比较推荐的是..."、"可能比较适合..."

### 避免用词
- [ ] "必买!"、"绝了!"、"太爱了!"
- [ ] "姐妹们冲!"、"一定要买!"
- [ ] "我不允许你还不知道!"

### 内容要求
- [ ] 真实体验分享（包含优缺点）
- [ ] 专业知识支撑
- [ ] 客观平衡评价
- [ ] 避免过度情绪化
- [ ] 避免夸张绝对化
- [ ] 避免虚假承诺

---

## 八、账号阶段策略

| 阶段 | 标题 | 结构 | 方向 | 商品 |
|------|------|------|------|------|
| 起号期(0-500粉) | 爆款型 | 黄金模板 | 颜值+求生 | 仅作案例提及，禁止价格 |
| 成长期(500-2000粉) | 混合 | 两种混合 | 根据内容选择 | 软植入，评论区引导，可提价格 |
| 成熟期(2000+粉) | 信任型 | 5步信任型 | 懒人+颜值 | 可挂链接，可说价格，可直接推荐 |

---

## 九、字数控制技巧

- [ ] 使用短句，避免长句
- [ ] 使用emoji代替部分文字
- [ ] 使用数字列表，提高信息密度
- [ ] 删除冗余的修饰词

**示例对比**：
- ❌ "在春秋季节的时候，我们需要大概每隔7-10天左右给绿萝浇一次水"（28字）
- ✅ "春秋季7-10天浇一次"（10字）
