# 小红书爆款内容战术

> 来源：《绿植园艺全域运营实操手册（2026完全版）》2.2-2.5节

## 双标题系统

**原理**：封面上的大标题（视觉重心）和底部的文案标题（搜索权重）承担不同职责。

| 标题类型 | 位置 | 功能 | 写作公式 |
|---------|------|------|----------|
| **视觉标题** | 图片/封面 | **吸引点击**（CTR） | 痛点+情绪+数字<br>（如"救命！我的琴叶榕叶子全掉了😭"） |
| **搜索标题** | 笔记底部 | **搜索排名**（SEO） | 核心关键词+长尾词堆砌<br>（如"室内绿植｜琴叶榕养护｜新手避坑"） |

## 钩子（Hook）设计

**常见爆款钩子**：
1. **利益前置**："这份《北京花市地图》我整理了3天，私信直接送..."
2. **制造焦虑**："2026年了，如果你还没..."
3. **身份认同**："如果你也是北京租房党..."

**强留粉钩子**：
- 文末必带"关注我，下期分享xxx"
- "收藏防走丢"

## 流量机制

**流量层级**：
- **初始池 (100-200)**：看封面点击率
- **千人池 (1000-5000)**：看互动率（转评赞）
- **万人池**：看完播率/停留时长

**流量低迷自救**：
- 检查违规：是否有引流违禁词（微信、加我）
- 二次发布：如果内容很好但流量差，修改封面和标题后重新发布

## 内容结构

```
开头（痛点共鸣）
  ↓
正文（解决方案）
  ├─ 要点1（问题 + 症状 + 解决方法）
  ├─ 要点2（问题 + 症状 + 解决方法）
  └─ 要点3（问题 + 症状 + 解决方法）
  ↓
结尾（商品推荐 + 行动引导）
```

**视觉呈现**：
- 每个要点配1张图
- 短段落（3行以内）
- 小标题 + 留白 + 分隔线
- 加粗重点 + emoji适当
