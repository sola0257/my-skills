# 小红书配图规范 v2.0

> **重要更新**：配图数量调整为12-18张，与文案段落匹配

---

## 📊 配图数量规范

### 总体要求
- **配图数量**：12-18张（含封面）
- **封面**：1张（带视觉标题文字）
- **正文配图**：11-17张
- **匹配原则**：每个段落/知识点至少1张配图

### 配图分配策略

**根据内容长度动态调整：**

| 内容类型 | 段落数 | 配图数量 | 说明 |
|---------|--------|---------|------|
| 短内容（3-5个要点） | 3-5段 | 12-14张 | 封面1张 + 每段2-3张 |
| 中等内容（5-8个要点） | 5-8段 | 14-16张 | 封面1张 + 每段2张 |
| 长内容（8-10个要点） | 8-10段 | 16-18张 | 封面1张 + 每段1-2张 |

---

## 🎨 配图类型与分配

### 1. 封面图（1张）
- **作用**：吸引点击，展示主题
- **要求**：
  - 比例：3:4 竖版
  - 叠加视觉标题文字
  - 画面要有冲击力
  - 体现核心主题

### 2. 开头引入图（1-2张）
- **作用**：营造氛围，引入主题
- **示例**：
  - 场景全景图
  - 痛点对比图
  - 问题展示图

### 3. 知识点配图（8-14张）
- **作用**：解释说明，增强理解
- **分配原则**：
  - 每个核心知识点：2-3张
  - 每个次要知识点：1-2张
  - 复杂步骤：每步1张

**配图类型：**
- **对比图**：前后对比、正确vs错误
- **步骤图**：分步展示操作过程
- **特写图**：细节展示、重点突出
- **场景图**：完整场景、氛围营造
- **示意图**：原理解释、结构说明

### 4. 总结/收尾图（1-2张）
- **作用**：总结要点，强化记忆
- **示例**：
  - 成果展示图
  - 要点汇总图
  - 互动引导图

## 📝 配图与文案匹配规则

### 匹配原则

**每张配图必须对应文案中的具体内容：**

1. **段落匹配**：每个段落至少1张配图
2. **要点匹配**：每个要点（✅ ① 等标记）对应1张配图
3. **步骤匹配**：每个操作步骤对应1张配图
4. **对比匹配**：文案中的对比内容用对比图展示

### 配图插入位置

**在Markdown文案中标注配图位置：**

```markdown
## 第一步：清理+规划 🧹

很多人一上来就想着买买买，其实应该先做减法。

![配图1](01.png)

清理清单：
✅ 枯死的植物（别舍不得，占地方还影响美观）
✅ 破损的花盆（换成统一风格的新盆）

![配图2](02.png)

空间规划：
把花园分成3个区域...

![配图3](03.png)
```

## 🎯 配图生成策略

### 内容分析阶段

1. **统计段落数**：计算正文有多少个段落/知识点
2. **确定配图数**：根据段落数确定需要12-18张中的具体数量
3. **分配配图**：为每个段落分配1-3张配图

### Prompt生成阶段

**为每张配图生成专属Prompt：**

```
配图1（封面）：主题全景
配图2（开头）：痛点场景
配图3（知识点1-1）：清理前的杂乱花园
配图4（知识点1-2）：清理后的整洁花园
配图5（知识点2-1）：蝴蝶兰特写
配图6（知识点2-2）：金桔特写
配图7（知识点2-3）：水仙特写
...
配图18（总结）：改造完成的花园全景
```

### 风格统一原则

**同一篇笔记的所有配图必须使用相同风格：**
- 选定1种风格（dreamy-photo / cozy-sketch / soft-botanical等）
- 所有配图使用相同的Prompt风格模板
- 保持色调、光线、氛围的一致性

## ⚠️ 注意事项

1. **配图不是越多越好**：
   - 12-18张是合理范围
   - 避免配图过多导致加载慢
   - 避免配图过少导致内容单调

2. **配图要有信息量**：
   - 每张图都要传达明确信息
   - 避免纯装饰性的无意义配图
   - 避免重复相似的配图

3. **配图要与文案紧密结合**：
   - 用户看图就能理解文案内容
   - 图文互补，不是简单重复
   - 配图位置要准确对应文案段落

4. **配图要符合小红书风格**：
   - 3:4竖版比例
   - 清晰、美观、有质感
   - 符合平台审美（莫兰迪色、氛围感）
