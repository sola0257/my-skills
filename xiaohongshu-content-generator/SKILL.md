## 📋 核心功能（v7.0）

### 1. **信任型双标题系统**

#### 视觉标题（封面大字）

**新公式：客观描述 + 价值点 + 时间/经验**

✅ **信任型标题示例：**
- 用了6个月的绿植养护心得，客观说说优缺点
- 3年花园设计经验：新手的5个常见误区
- 客观测评：这款网红植物到底值不值得买
- 踩过坑才明白：绿植养护的3个关键错误
- 真实体验：500元预算的阳台改造方案

❌ **避免的旧式标题：**
- 绝了！这个方法让我家绿植爆盆
- 姐妹们冲！绿植养护的秘密被我发现了
- 不允许你还不知道的神奇养护法

#### 搜索标题（笔记底部）

**保持不变：核心关键词 + 长尾词堆砌**

示例：`#绿植养护 #阳台花园 #居家绿植 #植物养护技巧 #室内植物 #园艺新手 #花园设计 #绿植避坑 #植物养护经验 #生活美学`

### 2. **信任型正文框架**

**5步信任型内容结构：**

```
1. 开头（建立共鸣+权威背景）
"最近很多朋友问我关于[主题]的问题，
我在[相关领域]做了[时间]，服务了[数量]客户，
今天客观分享一下我的经验和看法。"

2. 背景说明（增加可信度）
"其实我刚开始也踩了不少坑，
[具体经历]，后来通过[学习/实践]才找到正确方法。"

3. 核心内容（价值输出）
"总结了几个关键点，都是我踩过坑得出的经验：
第1个：[问题描述] + [解决方案] + [注意事项]
第2个：[问题描述] + [解决方案] + [注意事项]
第3个：[问题描述] + [解决方案] + [注意事项]"

4. 客观总结（平衡评价）
"整体来说，这个方法[总体评价]，
但也要注意[可能的不足]，
大家可以根据自己的情况选择。"

5. 建议指引（引导行动）
"如果大家有具体问题，
可以在评论区留言，我会逐一回复。
需要更详细资料的朋友，评论说'要资料'~"
```

### 5. **信任型配图策略**

#### 图片风格要求

**三大核心：**

1. **真实感强化**
   - 避免过度美化
   - 展示真实使用场景
   - 包含一些"不完美"细节

2. **专业感提升**
   - 清晰的技术细节图
   - 对比展示图
   - 步骤分解图

3. **客观性体现**
   - 优缺点对比图
   - 前后效果对比
   - 不同方案对比

## 🔄 执行流程

1. **解析输入**：获取主题、粉丝数、互动情况。
2. **生成文案**：
   - 生成信任型双标题（视觉标题+搜索标题）
   - 生成5步信任型正文框架
   - 生成 Tags
3. **违禁词检查**：调用 `compliance-checker`
4. **生成图片 Prompts**：
   - 封面 Prompt（真实感、专业感、客观性）
   - 正文配图 Prompts
5. **调用生成脚本** (`scripts/generate_xhs_post.py`)：
   - 生成底图
   - **合成封面**：将视觉标题叠加到底图上
   - 生成其他配图
6. **推荐发布时间**（v6.1保留）：
   - 基于小红书流量规律推荐最佳发布时间
   - 输出到 `发布信息.md`
7. **保存结果**：
   - Markdown 文档
   - 所有图片文件
   - 发布信息文件
